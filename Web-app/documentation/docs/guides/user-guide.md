# User Guide

## Getting Started

### Prerequisites
- A Google Account
- Text to be formatted

### How to Use

1.  **Login**: Visit the application and sign in with your Google Account.
2.  **Dashboard**: Once logged in, you will see the dashboard.
3.  **Upload/Paste**: Paste your unstructured text into the input field or upload a supported file.
4.  **Select Template**: Choose from the available academic templates (e.g., College Report, Thesis, Assignment).
5.  **Format**: Click "Format". The AI will process your document.
6.  **Export**: Once complete, the formatted document will be available in your Google Docs using your connected account.
